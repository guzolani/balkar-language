# Балкарский

Мобильное веб-приложение для изучения балкарского варианта карачаево-балкарского языка.

## Запуск

Приложение не требует сборки или внешних зависимостей:

```bash
python3 -m http.server 4173
```

После запуска откройте `http://localhost:4173`.

## Что реализовано

- адаптивная главная страница;
- первый урок из семи фраз;
- карточки с самооценкой ответа;
- словарь с поиском и тематическими фильтрами;
- локальное сохранение прогресса;
- светлая и тёмная темы;
- маркировка непроверенных материалов и источников.
- установка на главный экран iPhone как PWA без App Store;
- офлайн-кэширование интерфейса и урока;
- автоматическое сохранение прогресса на устройстве;
- экспорт и восстановление резервной копии прогресса.

## Источник демонстрационных данных

Русско-карачаево-балкарский словарь-разговорник, Черкесск, 1990. Все записи демонстрационного урока помечены как ожидающие проверки носителем.
