# Балкарский: курс по двум главам

Мобильное PWA для последовательного изучения материалов «ГЛАВА 1.pdf» и «ГЛАВА 2.pdf» из подключённого Google Drive.

## Запуск

Приложение не требует сборки или внешних зависимостей:

```bash
python3 -m http.server 4173
```

После запуска откройте `http://localhost:4173`.

## Проверка

```bash
node --check app.js
node --check chapter-data.js
node tests/course.test.mjs
```

## Структура курса

- ГЛАВА 1: 18 уроков, от алфавита и гармонии до падежей;
- ГЛАВА 2: 16 уроков по временам, наклонениям и залогам;
- hash-маршруты для прямых ссылок на главы и уроки;
- сохранение прохождения и попыток в `localStorage`;
- упражнения только для однозначных пар, вручную подтверждённых по источнику;
- итоговое повторение из пяти заданий после каждой главы;
- понятное объяснение каждого урока: простое определение, правило по шагам, разобранные примеры и памятка;
- подробный читабельный материал вместо сырого OCR, с балкарским примером на каждом шаге;
- игровая практика из семи заданий с очками, сериями и объяснением ошибок;
- сборка предложений из перемешанных слов с возможностью возвращать и переставлять слова;
- академический блок: функция формы, модель образования, ограничения и отличие от соседних тем;
- отдельная вводная по 37 буквам алфавита и вручную связанные пары «правило → балкарский пример»;
- адаптивная и клавиатурно-доступная навигация;
- PWA и офлайн-кэш.
